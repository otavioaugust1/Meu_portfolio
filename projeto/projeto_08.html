Consolidação Portaria 90

$ python main.py --input data/raw/ --output data/results/

Automação completa para consolidação de dados da Portaria GM/MS nº 90/2023, unificando planilhas e gerando relatórios para apoio à gestão do SUS.

Ver no GitHub Portaria Oficial Voltar

O Problema

A Portaria GM/MS nº 90, de 6 de fevereiro de 2023, estabelece diretrizes para organização e funcionamento das redes de atenção à saúde no SUS. O acompanhamento de sua execução envolve grandes volumes de dados distribuídos em múltiplas planilhas e competências.

Este projeto automatiza todo o processo de consolidação dos dados: une as planilhas brutas de diferentes competências, aplica as transformações necessárias e gera relatórios consolidados prontos para análise e tomada de decisão.

Fluxo de Processamento

1. Dados Brutos

Planilhas por competência colocadas em data/raw/

2. Processamento

Limpeza, padronização e transformação com Pandas

3. Consolidação

Junção de todas as competências em um único dataset

4. Relatório

Saída em data/results/ pronta para análise

Funcionalidades

Consolidação de Múltiplas Planilhas

Une automaticamente todos os arquivos da pasta de entrada

Geração de Relatórios

Exporta resultado consolidado em formatos prontos para uso

Arquitetura Modular

Separação em controller, models e views para fácil manutenção

Conformidade com a Portaria

Validação dos dados conforme diretrizes da GM/MS nº 90/2023
