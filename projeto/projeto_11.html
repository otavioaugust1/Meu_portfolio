$ python bot_monitoramento.py --componente geral --modo automatico

Robô de automação para monitorar, coletar e consolidar dados do Componente Especializado da Assistência Farmacêutica (CEAF) no SUS, com alertas e relatórios automáticos.

Ver no GitHub Voltar aos Projetos

Visão Geral do Projeto

O Componente Especializado da Assistência Farmacêutica (CEAF) é responsável pelo fornecimento de medicamentos de alto custo no SUS. O monitoramento manual desses dados é trabalhoso e sujeito a erros — este BOT resolve esse problema com automação completa.

O robô acessa portais do DATASUS e sistemas do Ministério da Saúde, coleta dados de dispensação, consolida por competência e gera painéis de monitoramento automáticos com alertas de inconsistências.

Funcionalidades

Coleta Automatizada

Navegação automática nos portais do DATASUS com Selenium para extração de dados de dispensação por competência.

Alertas de Inconsistências

Detecção e notificação automática de divergências entre dados esperados e reportados pelos estados.

Monitoramento de Medicamentos

Acompanhamento por grupo de medicamento, CID, UF e estabelecimento, com histórico temporal.

Relatórios Consolidados

Geração automática de planilhas e dashboards prontos para uso pelos gestores técnicos do MS.

Stack Técnica

Python Selenium Pandas OpenPyXL Schedule DATASUS

O Componente Especializado representa bilhões de reais em medicamentos anuais. A automação do monitoramento reduz o tempo de análise de dias para minutos, permitindo respostas mais rápidas a irregularidades e garantindo o acesso contínuo aos medicamentos pelos pacientes do SUS.
