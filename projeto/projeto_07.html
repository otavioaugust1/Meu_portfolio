Sobre o Bootcamp

O Bootcamp DIO Unimed-BH Ciência de Dados é uma jornada educacional completa voltada para profissionais que atuam ou desejam ingressar na área de Data Science. O programa une fundamentos teóricos com aplicações práticas reais.

Este repositório consolida todos os desafios de código, projetos práticos e notebooks desenvolvidos ao longo do programa, incluindo o projeto final de Machine Learning para previsão da evolução do COVID-19 no Brasil.
