BOT CAPTCHA

Reconhecimento e resolução automática de CAPTCHAs usando visão computacional e rede neural convolucional treinada com TensorFlow/Keras.

Ver no GitHub Voltar aos Projetos

Visão Geral

Este projeto utiliza visão computacional e aprendizado de máquina para reconhecer e resolver CAPTCHAs — aqueles testes visuais usados em formulários para provar que o usuário é humano.

O bot passa por todo o pipeline: desde o tratamento da imagem bruta, separação dos caracteres, treinamento da rede neural convolucional (CNN), até a resolução automática dos CAPTCHAs em produção. Inspirado nas aulas práticas da Hashtag Programação, o projeto consolida aprendizados de CV e deep learning aplicados a automação real.

Pipeline do Bot

1. Tratamento

Converte para escala de cinza, remove ruídos e fundo da imagem CAPTCHA

Bot_imagem.py

2. Separação

Isola cada caractere da imagem e organiza em pastas por letra

Bot_separar.py

3. Treinamento

Treina CNN com TensorFlow/Keras. Modelo salvo em .hdf5

Bot_treinar.py

4. Resolução

Usa o modelo treinado para resolver CAPTCHAs automaticamente

Bot_resolver.py

Funcionalidades

Pré-processamento de imagem

Escala de cinza, binarização, remoção de ruído

Segmentação de caracteres

Contour detection para isolar cada letra/número

CNN com TensorFlow/Keras

Rede neural convolucional para classificação de caracteres

Modelo persistente

Modelo treinado salvo em .hdf5 reutilizável

Automação completa

Do download ao preenchimento automático do CAPTCHA

Renomeação automática

Organização do dataset para treinamento futuro
