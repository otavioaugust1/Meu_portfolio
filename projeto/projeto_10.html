Visão Geral do Projeto

O DoctorCare é uma plataforma web voltada para saúde digital, com foco em acessibilidade e experiência do usuário. O projeto apresenta uma landing page moderna para uma clínica médica, com seções de especialidades, depoimentos, planos de atendimento e agendamento.

Desenvolvido com HTML, CSS avançado e JavaScript puro, prioriza performance, design responsivo e boas práticas de acessibilidade — servindo como base para soluções de saúde digital no contexto do SUS e privado.

Funcionalidades

Design Responsivo

Layout adaptado para mobile, tablet e desktop com CSS Grid e Flexbox modernos.

Catálogo de Especialidades

Apresentação visual das especialidades médicas disponíveis com ícones e descrições objetivas.

Agendamento Online

Seção de agendamento simplificado com formulário acessível e feedback visual imediato.

Modo Escuro

Alternância entre tema claro e escuro com persistência via LocalStorage.

Stack Técnica

HTML5 CSS3 JavaScript CSS Grid Flexbox LocalStorage

O DoctorCare demonstra como soluções web modernas podem ser aplicadas na saúde para melhorar a experiência do paciente — desde o primeiro contato até o agendamento. Serve como protótipo replicável para clínicas, cooperativas médicas e integrações com sistemas como o e-SUS.
