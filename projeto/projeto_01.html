Analisador de Filas de Espera

$ python analisador_filas.py --setor saude --output relatorio

Ferramenta desenvolvida para auxiliar gestores de saúde na análise das filas de espera e identificação de gargalos no acesso a serviços de saúde pública.

Ver no GitHub Voltar aos Projetos

Visão Geral do Projeto

As filas de espera por serviços de saúde representam um dos maiores desafios do SUS. Este aplicativo foi desenvolvido para auxiliar gestores estaduais e municipais na análise dessas filas, identificando gargalos e apoiando a tomada de decisão baseada em dados.

A ferramenta processa dados de filas de espera, gera relatórios gerenciais e apresenta indicadores estratégicos para priorização do atendimento.

Funcionalidades

Importação de Dados

Suporte a arquivos CSV e XLSX com dados de filas de espera dos sistemas municipais e estaduais.

Análise Estatística

Geração automática de indicadores: tempo médio de espera, volume por especialidade, priorização por urgência.

Relatórios Gerenciais

Exportação de relatórios estruturados para apoio à gestão e prestação de contas.

Geolocalização

Mapeamento geoespacial dos dados para identificação de regiões críticas com alta demanda reprimida.

Stack Técnica

Python Pandas NumPy Matplotlib Streamlit Geopandas OpenPyXL

Projeto desenvolvido no contexto da OCI (Organização de Cuidados Integrados), com foco real na gestão das listas de espera do SUS. A ferramenta foi prototipada para auxiliar estados e municípios na tomada de decisão baseada em dados.
