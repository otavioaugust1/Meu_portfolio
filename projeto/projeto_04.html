Automação do download, tratamento e consolidação dos dados do painel INVESTSUS — Crédito Financeiro e Modalidade 1 do Ministério da Saúde do Brasil.

Ver no GitHub Voltar aos Projetos

Visão Geral do Projeto

A Portaria GM/MS nº 90 regulamenta os critérios de transferência de recursos financeiros para estados e municípios no âmbito do SUS. Monitorar esses dados manualmente é um processo trabalhoso e sujeito a erros.

Este bot automatiza todo o pipeline: acessa o painel INVESTSUS, faz o download dos dados de Crédito Financeiro e Modalidade 1, realiza o tratamento e entrega dados consolidados prontos para análise.

Pipeline de Automação

1. Acesso Web
Navegação automática ao painel INVESTSUS
2. Download
Extração automática dos arquivos de dados
3. Tratamento
Limpeza, padronização e enriquecimento
4. Consolidação
Dados prontos para análise e relatórios

Dados Tratados

Crédito Financeiro

Valores transferidos pela União para estados e municípios via Fundo Nacional de Saúde, organizados por período, UF, município e bloco de financiamento.

Modalidade 1

Dados de repasses na modalidade fundo a fundo, com detalhamento por ação orçamentária, programa e grupo de despesa, facilitando o controle social e a transparência.
