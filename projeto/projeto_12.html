API-REST — Integração SISREG

$ curl -X GET https://api.sisreg.sus.gov.br/v1/filas?uf=DF&especialidade=cardio

Interface de Programação de Aplicação (API) RESTful para integração entre sistemas de regulação do SUS, possibilitando acesso padronizado a dados do SISREG e eliminando retrabalho entre secretarias estaduais e municipais.

Visão Geral do Projeto

A API-SISREG é um projeto de interoperabilidade que expõe dados do Sistema Nacional de Regulação (SISREG) através de endpoints RESTful padronizados. Permite que sistemas externos — municipais, estaduais e privados — consultem filas de espera, ofertas de vagas e indicadores de regulação de forma programática.

O projeto segue os padrões de interoperabilidade do e-SUS e RNDS (Rede Nacional de Dados em Saúde), com autenticação segura via token e documentação automática via Swagger/OpenAPI.

Principais Endpoints

GET /v1/filas?uf={uf}&especialidade={codigo}

Consulta filas de espera por UF e especialidade

GET /v1/oferta-vagas/{municipio_ibge}

Lista oferta de vagas disponíveis por município

POST /v1/solicitacoes

Envia nova solicitação de regulação ao SISREG

GET /v1/indicadores/tempo-espera

Retorna indicadores agregados de tempo de espera por especialidade

Arquitetura & Funcionalidades

Autenticação Segura

Tokens JWT com expiração, controle de escopo por perfil (município, estado, MS) e rate limiting.

Documentação Swagger

Documentação automática e interativa via OpenAPI 3.0, com exemplos de requisição e resposta.

Interoperabilidade

Compatível com padrões RNDS, FHIR e SISREG, facilitando integração com sistemas já existentes.

Cache & Performance

Cache Redis para consultas pesadas, garantindo respostas rápidas mesmo com alto volume de requisições.

Stack Técnica

Python FastAPI SQLAlchemy PostgreSQL Redis JWT OpenAPI 3.0 Docker

A interoperabilidade é um dos principais gargalos do SUS digital. A API-SISREG atua como ponte entre o sistema federal de regulação e os sistemas locais, reduzindo duplicação de dados e acelerando o acesso à informação para tomada de decisão em saúde.
