Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar meus principais projetos, publicações e informações profissionais de forma interativa e visualmente atraente.

## 🌟 Visão Geral

Este portfólio foi criado utilizando tecnologias web modernas, com foco em design responsivo e experiência do usuário. Ele inclui:
- **Seção de Projetos:** Detalhes sobre meus principais projetos, com links para páginas individuais e repositórios no GitHub.
- **Seção Sobre Mim:** Informações sobre minha trajetória profissional e acadêmica.
- **Publicações:** Lista de artigos, manuais e apresentações técnicas que produzi.
- **Dark Mode:** Tema escuro para uma experiência de visualização confortável.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Design responsivo com variáveis CSS e animações.
- **JavaScript:** Funcionalidades interativas, como animações e integração com a API do GitHub.
- **GitHub Pages:** Hospedagem do portfólio.

## 📂 Estrutura do Projeto

```
Meu_portfolio/
├── index.html          # Página inicial
├── asset/
│   ├── projetos.html   # Página com todos os projetos
│   ├── sobre.html      # Página sobre mim
├── projeto/            # Páginas individuais dos projetos
│   ├── projeto_01.html
│   ├── projeto_02.html
│   └── ...
├── CSS/
│   ├── style.css       # Estilos principais
│   ├── tab.css         # Estilos adicionais
├── script/
│   └── script.js       # Funcionalidades interativas
├── img/                # Imagens utilizadas
├── README.md           # Este arquivo
└── LICENSE             # Licença do projeto
```

## 🚀 Funcionalidades

- **Dark Mode:** Tema escuro ativado automaticamente ou manualmente pelo usuário.
- **Animações:** Efeitos de scroll-reveal e hover para maior interatividade.
- **Integração com GitHub:** Exibição dinâmica de repositórios utilizando a API do GitHub.
- **Design Responsivo:** Compatível com dispositivos móveis, tablets e desktops.
